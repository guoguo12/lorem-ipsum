Introduction

This is the help page for the Lorem Ipsum add-on for Google Docs.

Using the Add-on

After you install the add-on, click Add-ons > Lorem Ipsum > Insert placeholder text... to add placeholder text to your document. You can add up to 99 randomly generated words, sentences, or paragraphs.
